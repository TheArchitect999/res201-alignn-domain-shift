# From-Scratch Training Curves

These figures were generated from the saved `history_train.json` and `history_val.json` files for every completed from-scratch run in this namespace.

- results root: `Results_Hyperparameter_Set_1`
- run subdirectory: `train_alignn_fromscratch`

Each per-run figure contains:
- left panel: train and validation L1 loss curves
- right panel: train and validation relative error reduction from epoch 1

Files in this folder:
- one PNG and one PDF per from-scratch run
- one grid figure per family for quick browsing
- `training_curve_manifest.csv` with file paths and run metadata

Family grid figures:
- oxide: `reports/Hyperparameter Set 1/Training Curves/From Scratch/oxide_training_curve_grid.png` and `reports/Hyperparameter Set 1/Training Curves/From Scratch/oxide_training_curve_grid.pdf`
- nitride: `reports/Hyperparameter Set 1/Training Curves/From Scratch/nitride_training_curve_grid.png` and `reports/Hyperparameter Set 1/Training Curves/From Scratch/nitride_training_curve_grid.pdf`
